# 说明
本工具是windows下打包 tgz文件并上传到Tars平台的工具
新手作品,欢迎大佬提意见

# 安装
开启 go module
在项目内执行
`go get github.com/Swmiao1/tarstools`
# 使用
详细参数可以`tarstools -h` 查看
第一次使用请运行 `tarstools` 生成配置文件

###上传并发布
在项目目录下打开 `cmd`  或 在`IDE`中打开 `Terminal` 工具
输入 `tarstools -u -p`

###token获取方法
打开此页面 `你的tars地址/auth.html#/token`
点击新增Token
